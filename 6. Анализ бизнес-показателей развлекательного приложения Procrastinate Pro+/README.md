# Анализ бизнес-показателей развлекательного приложения Procrastinate Pro+

Продукт - развлекательное приложение Procrastinate Pro+. Несколько прошлых месяцев бизнес постоянно нес убытки - в привлечение пользователей была вложено большое количество денег без ощутимого результата. Необходимо определить причины неэффективности привлечения пользователей и сформировать рекомендации для отдела маркетинга для повышения эффективности 

## Задачи проекта
* Анализ продукта, продаж и маркетинговых вложений  
* Анализ ключевых бизнес-метрик  
* Когортный анализ   
* Рекомендации по оптимизации маркетинговых затрат  

## Используемые библиотеки
*Pandas*, *Matplotlib*, *Seaborn*

## Результаты исследования
Проанализировано поведение пользователей: какими устройствами пользуются пользователи, из каких стран и по каким каналам привлечены пользователи.   
  
Проведен когортный анализ по ключевым бизнес-метрикам — средний чек (одного заказа и одного покупателя), LTV, CAC, ROMI, RR и CR, а также DUA, WAU и MUA 
  
Расчитан срок окупаемости вложений в маркетинг, а также даны рекомендации для корректировки маретинговой стратегии продвижения сервиса.
